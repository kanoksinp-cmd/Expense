# Update Log — เพิ่มปุ่มเปิด/ปิด Sidebar

## การเปลี่ยนแปลงในเวอร์ชันนี้

### ✨ ฟีเจอร์ใหม่

**1. ปุ่มเปิด/ปิดเมนูข้าง (Sidebar Toggle)**
- ปุ่มสีน้ำเงินลอยอยู่มุมซ้ายบน
- กดเพื่อซ่อน/แสดงเมนูข้างได้
- ระบบจดจำสถานะ (เปิด/ปิด) แม้ปิดแล้วเปิดใหม่

**2. รองรับการใช้งานบนมือถือ**
- หน้าจอมือถือ: เมนูซ่อนอัตโนมัติ เปิดด้วยปุ่ม
- หน้าจอ Desktop: เมนูแสดงตามปกติ สามารถซ่อนเพื่อขยายพื้นที่การทำงาน
- มี overlay สีเทาบนมือถือเมื่อเมนูเปิด กดนอกเมนูเพื่อปิด

**3. Animation นุ่มนวล**
- Transition 0.3 วินาที เมื่อเปิด/ปิด
- Hover effect บนปุ่ม
- ไอคอนเปลี่ยนระหว่าง ☰ และ ✕ บนมือถือ

### 🔧 การปรับปรุง

**Desktop (จอกว้างกว่า 768px):**
- ปุ่มอยู่ซ้ายบน ข้างเมนู (เลื่อนตามเมนู)
- กดปุ่ม → เมนูซ่อน + พื้นที่ทำงานขยายเต็มจอ
- เหมาะสำหรับคนต้องการพื้นที่กว้างดูข้อมูล

**Mobile (จอแคบกว่า 768px):**
- เมนูซ่อนอัตโนมัติตั้งแต่เริ่มต้น
- กดปุ่ม → เมนูเลื่อนออกมาจากซ้าย + มี overlay
- กดนอกเมนูหรือกดปุ่มอีกครั้งเพื่อปิด
- Grid แบ่ง 2 คอลัมน์จะกลายเป็น 1 คอลัมน์อัตโนมัติ

### 📱 ขนาดหน้าจอที่รองรับ

- **Mobile:** < 768px
- **Tablet/Desktop:** ≥ 768px
- ระบบปรับ layout อัตโนมัติตาม breakpoint

### 💾 ไฟล์ที่อัปเดต

- `index.html` — เพิ่ม CSS, HTML button, และ JavaScript functions
  - CSS: sidebar toggle, overlay, responsive media queries
  - HTML: toggle button + overlay element
  - JS: `toggleSidebar()`, `closeSidebarMobile()`, `restoreSidebarState()`
- `README.md` — เพิ่มรายละเอียดฟีเจอร์ใหม่

### 🚀 วิธีใช้งาน

1. **Desktop:** กดปุ่มซ้ายบนเพื่อซ่อน/แสดงเมนู
2. **Mobile:** กดปุ่มเพื่อเปิดเมนู กดนอกเมนูหรือปุ่มอีกครั้งเพื่อปิด
3. สถานะจะจดจำอัตโนมัติ ไม่ต้องตั้งค่าใหม่ทุกครั้ง

### ⚙️ การ Deploy

ไม่ต้องเปลี่ยนการตั้งค่าบน Render.com
